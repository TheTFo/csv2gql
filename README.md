# CSV2GQL

Imports a CSV file to a GraphQL API.

```
Usage: csv2gql [options] [command]

Options:
  -V, --version                                output the version number
  -h, --help                                   display help for command

Commands:
  import|i [options] <csv-file> <graphql-uri>  Import a CSV file to the GraphQL API
  help [command]                               display help for command
```
